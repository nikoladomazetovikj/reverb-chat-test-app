# Laravel Rever Chat App Testing


## Setup 

Install `npm` packages:

```bash
npm install
```

Create `.env` file
```bash
cp .env.example .env
```

Setup   reverb` configuration in the `.env` file 

```dotenv
VITE_CHAT_ID=your_channel_name
VITE_BEARER=your_auth_bearer_token
VITE_BASE_URL=api_server_base_url
```
